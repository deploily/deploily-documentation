# 🎓 Guide Utilisateur : Créer un Cours avec le Module E-Learning (Odoo 18)

Ce guide explique pas à pas comment un utilisateur peut créer et publier un cours en ligne avec le module **E-learning** d’Odoo 18.

---

## 🚀 Étape 1 : Installer le module
1. Connectez-vous à votre base Odoo 18.  
2. Allez dans **Applications (Apps)**.  
3. Recherchez **E-learning** et cliquez sur **Installer**.  

Une nouvelle application **E-learning** apparaîtra dans le menu principal.

---

## 📝 Étape 2 : Créer un nouveau cours
1. Cliquez sur le menu **E-learning**.  
2. Dans la vue **Cours**, cliquez sur **Créer** (**Nouveau**).  
3. Remplissez les informations principales :
   - **Nom du cours** : Exemple *Python pour Débutants*  
   - **Catégorie** : Choisissez ou créez une catégorie (ex. Informatique, Formation interne).  
   - **Description** : Ajoutez une brève description du cours.  
   - **Responsable** : Cliquer sur **Options** et selectionner  La personne en charge du cours.  

---

## 🔑 Étape 3 : Définir la visibilité
Dans l’onglet **Options** du cours, configurez la visibilité :

- **Public** → Tout le monde peut voir et s’inscrire.  
- **Utilisateurs connectés** → Réservé aux personnes avec un compte Odoo.  
- **Membres du cours uniquement** → Seules les personnes invitées ou inscrites manuellement.  

👉 Choisissez selon vos besoins (par ex. **Public** pour une formation ouverte).

---

## 📂 Étape 4 : Ajouter du contenu
1. Ouvrez le cours et cliquez sur **Ajouter du contenu**.  
2. Choisissez :
   - **Section** → Organiser les leçons en chapitres/modules.  
   - **Contenu** → Ajouter une ressource (vidéo, document, texte, etc.).  

### Types de contenu possibles :
- **Vidéo** : Intégration YouTube, Vimeo, ou upload direct.  
- **Document/PDF** : Support de cours ou tutoriel.  
- **Texte** : Articles ou guides écrits.  
- **Quiz** : Exercices interactifs.  

Exemple :  
- Section : *Introduction*  
- Contenu : *Vidéo de présentation*  

---

## 🧩 Étape 5 : Créer un quiz (optionnel)
1. Dans une section, cliquez sur **Ajouter → Quiz**.  
2. Ajoutez des questions :
   - **Choix multiple**  
   - **Vrai/Faux**  
   - **Réponse unique**  
3. Définissez :
   - **Score minimum requis** pour réussir.  
   - **Nombre de tentatives autorisées**.  
   - **Certification** (si vous souhaitez délivrer un certificat après réussite).  

---

## 👥 Étape 6 : Gérer les participants
- Dans l’onglet **Participants**, vous pouvez :
  - Ajouter des apprenants manuellement.  
  - Autoriser les inscriptions libres (selon visibilité).  
  - Suivre la progression et les résultats.  

---



---

# 🎯 Conclusion
Avec ce processus, vous pouvez créer un cours complet dans Odoo 18 E-learning, choisir qui peut y accéder, enrichir le contenu avec vidéos et quiz, et suivre la progression de vos apprenants.
