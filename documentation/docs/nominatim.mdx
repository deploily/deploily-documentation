**Nominatim**

The Nominatim API is a web-based geocoding service that provides forward and reverse geocoding capabilities using OpenStreetMap (OSM) data. It allows users to convert addresses or place names into geographic coordinates and retrieve detailed location information from latitude and longitude inputs.
